# React expensify app
This app used for manage expenses. Separate environment  rebase database for testing.

#### Installation

1. Install this code on your local system
    ```
    Clone this project
   
2. Change into directory

    ```
    cd react-expensify-app
    ```
    
3. Install dependencies

    ```
    npm install
    ```

5. Start project

    ```
    npm run start
